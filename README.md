# LaunchMind Startups

This repository is managed by the [LaunchMind](https://github.com/muhammadhaider02/LaunchMind) Engineer Agent.

## Workflow

1. A new startup idea is generated by the multi-agent system.
2. The agent creates a branch: `startup/<startup-name>`.
3. A landing page for the startup is added in its folder.
4. A Pull Request is automatically created.
5. Merge the PR to publish the startup.
